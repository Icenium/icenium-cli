To debug Apache Cordova apps on iOS devices, complete the following steps:

  1. In the main menu bar, select Develop.

    If you do not see the Develop menu, you need to enable the Web Inspector. For more information, see Safari Web Inspector Guide.

  2. Choose your device from the list.

  3. From the context menu for the device, select the view that you want to debug.

  4. Start debugging your app. For more information about how to use the Safari developer tools, see Web Development Tools.
